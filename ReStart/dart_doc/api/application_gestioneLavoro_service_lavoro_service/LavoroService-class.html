LavoroService

LavoroService class abstract

Interfaccia per il service di gestione degli annunci di lavoro.

Definisce i metodi per la manipolazione e il recupero degli annunci di lavoro, incluso l'aggiunta, la modifica, l'approvazione, il rifiuto e la visualizzazione degli annunci.
